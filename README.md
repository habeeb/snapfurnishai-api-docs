# snapfurnishai-api-docs
Interactive OpenAPI documentation for SnapFurnish AI. Built with an open-source mindset to enable seamless third-party agentic integrations and commerce workflows.

🛠 Project Maturity: Experimental Prototype (Alpha)
This repository hosts the API specification for the SnapFurnish AI capstone project. It is currently in a "Prototype" state.

What’s missing for Production-Readiness:

Rate Limiting: Currently lacks DDoS protection at the API Gateway level.

Auth/Security: Uses mocked authentication; requires Authentication to be implemented for production.

Observability: Basic logging only; requires full OpenTelemetry/Langfuse instrumentation for scale.
